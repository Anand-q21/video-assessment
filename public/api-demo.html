üöÄ Quick Start

Get Started in 3 Steps:

  1. Start the server: symfony server:start
  2. Register a user: Use the registration endpoint below
  3. Get access token: Login to receive JWT tokens
cURL
Postman
Swagger UI
# Register a new user curl -X POST http://localhost:8000/api/register \ -H "Content-Type: application/json" \ -d '{ "email": "demo@example.com", "username": "demouser", "firstName": "Demo User", "password": "securepassword123" }' # Login to get access token curl -X POST http://localhost:8000/api/login \ -H "Content-Type: application/json" \ -d '{ "email": "demo@example.com", "password": "securepassword123" }'

Import the Postman collection: Video_Platform_API.postman_collection.json

Set environment variables:

  • baseUrl: http://localhost:8000/api
  • accessToken: (will be set automatically after login)

Visit the interactive API documentation:

http://localhost:8000/api/docs

Use the "Authorize" button to set your JWT token for authenticated requests.
